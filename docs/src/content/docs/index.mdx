## Install and get started now

<Steps>

1.  **Install the module**

    First, install module `one` command into your path:

    <Tabs group="pkgmanager">

        <TabItem label="npm">

        	```sh title="Install via npm"
        	npm i
        	```

        </TabItem>

        <TabItem label="yarn">

        	```sh title="Install via Yarn"
        	yarn
        	```

        </TabItem>

        <TabItem label="pnpm">

        	```sh title="Install via pnpm"
        	pnpm install 
        	```

        </TabItem>

    </Tabs>

1.  **Create your bot**
   use the `startup` command to run your bot in panel

    ```sh title="startup command"
    node main.js --pairing-code --autocleartmp --autoread
    ```

1.  **Keep going…**

    You're nearly done. Follow the in-prompt instructions or continue on with the [getting started guide](/docs/getting-started/).

</Steps>
